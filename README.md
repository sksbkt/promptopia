
# 🚀 Promptopia

> **Discover & Share AI-Powered Prompts**

Promptopia is a modern full-stack web application that enables users to create, discover, edit, and share AI prompts with the community. Built with **Next.js**, **MongoDB**, **NextAuth**, and **Tailwind CSS**, it provides secure authentication, a responsive interface, and a seamless user experience.

## 🌐 Live Demo

👉 https://promptopia-quantic.vercel.app/

---

## ✨ Features

- 🔐 Google Authentication with NextAuth
- 📝 Create, edit, and delete AI prompts
- 🌍 Browse prompts shared by the community
- 👤 Personalized user profiles
- 🔎 Search prompts by username, tag, or prompt content
- 🏷️ Clickable tags for quick filtering
- ⚡ Optimized performance with Next.js App Router
- 📱 Fully responsive design
- ☁️ MongoDB database integration
- 🎨 Clean and modern UI with Tailwind CSS

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/sksbkt/promptopia.git
```

### Navigate to the project

```bash
cd promptopia
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```
